## 📝 支持的Markdown语法

### 标题
```markdown
# 一级标题（小三号，居中加粗）

## 二级标题（小四加粗，首行缩进）

### 三级标题（小四不加粗，首行缩进）
```

### 段落和列表
```markdown
这是一个段落，会自动应用：
- 仿宋_GB2312字体
- 小四字号
- 1.5倍行距
- 首行缩进2字符

1. 有序列表项
2. 第二项
3. 第三项

- 无序列表项
- 第二项
```

### 表格
```markdown
| 项目 | 要求 | 备注 |
|------|------|------|
| 高质量 | 必须 | 强制 |
| 标准 | 推荐 | 建议 |
```

### 引用
```markdown
> 这是一个引用块
> 显示为灰色背景
```

### 文本格式
```markdown
**加粗文本**
*斜体文本*
~~删除线文本~~
```

## 📊 格式输出特性

转换后的文档包含：

1. **标准法律文书格式**
   - 仿宋_GB2312字体
   - 小四号正文
   - 小三号一级标题

2. **专业排版**
   - 1.5倍行距
   - 首行缩进2字符
   - 两端对齐

3. **页面设置**
   - A4纸张 (21cm × 29.7cm)
   - 上下边距：2.54cm
   - 左右边距：3.18cm

4. **自动附加**
   - 页脚页码（1/x格式）
   - 甲乙双方签字区域
   - "【以下无正文】"标记

## ⚠️ 注意事项

1. **输入文件编码**：确保Markdown文件为UTF-8编码
2. **docx依赖**：需要安装`npm install docx`
3. **权限**：脚本已有执行权限，无需额外设置
4. **路径**：在Claude Code中调用时会自动处理路径

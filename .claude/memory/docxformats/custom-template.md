# 自定义格式模板

> Last updated: 2025-11-13
> 本文件提供了一个完整的自定义格式配置模板，您可以复制并修改为符合您个人偏好的格式设置

## 如何使用本模板

### 步骤1: 复制模板
1. 复制本文件到您的 `.claude/memory/docxformats/` 目录
2. 重命名为 `my-custom.md`（或您喜欢的名称）

### 步骤2: 修改配置
1. 根据您的需求调整各项设置
2. 保存文件

### 步骤3: 使用自定义格式
```bash
/2word --format=my-custom input.md output.docx
```

## 注意事项

1. **格式名称**: 使用英文和数字，避免特殊字符
2. **测试**: 建议先用简单文档测试格式效果

## 常见问题

### Q: 如何删除自定义格式？
A: 直接删除 `.claude/memory/docxformats/your-format.md` 文件即可。

### Q: 如何恢复默认格式？
A: 使用 `/2word --format=default` 命令。

### Q: 自定义格式不生效怎么办？
A: 检查格式文件是否在正确目录，格式名称是否正确。

### Q: 可以同时使用多个自定义格式吗？
A: 可以，每个律师可以维护多套格式配置。

## 答辩状模板

### 基本结构

```
{[案件基本信息]}
{[答辩请求]}
{[事实和理由]}
{[证据目录]}
{[结尾]}
```

### 详细模板

#### 案件基本信息
```
案件性质：[民事/刑事/行政]案件
案件编号：[案号]
受理法院：[法院名称]
答辩人（被告）：[姓名/名称]
住所地：[地址]
法定代表人：[姓名/职务]
委托代理人：[姓名，律师事务所]
```

#### 答辩请求
```
答辩人针对原告的起诉，答辩如下：

1. 请求法院依法驳回原告的全部诉讼请求；

2. 本案诉讼费用由原告承担；

3. [其他请求，如反诉等]
```

#### 事实和理由

##### 一、对案件事实的答辩
```
[根据案件具体情况，详细陈述被告方的事实主张]

1. 关于争议事实一的说明：
[具体事实陈述]

2. 关于争议事实二的说明：
[具体事实陈述]
```

##### 二、对原告诉讼请求的答辩
```
1. 对原告第一项诉讼请求的答辩：
[具体答辩理由]

2. 对原告第二项诉讼请求的答辩：
[具体答辩理由]
```

##### 三、法律适用分析
```
根据《[相关法律名称]》第[条款]规定：
[法律条文引用]

本案中，[法律分析]，因此原告的诉讼请求缺乏法律依据。
```

##### 四、争议焦点的意见
```
1. 争议焦点一：[焦点名称]
答辩意见：[具体意见]

2. 争议焦点二：[焦点名称]
答辩意见：[具体意见]
```

#### 证据目录
```
答辩方提交的证据如下：

证据一：[证据名称]
证据来源：[来源说明]
证明目的：[证明什么]

证据二：[证据名称]
[同样格式]
```

#### 结尾
```
此致
[法院名称]

答辩人：[签字]
[日期]
```

### 注意事项

1. **逻辑清晰**：事实、理由、法律分析层次分明
2. **针对性强**：逐项回应原告的诉讼请求
3. **证据支撑**：所有主张都要有证据支撑
4. **语言规范**：使用标准法律术语，表述准确

## 起诉状模板

### 基本结构

```
{[当事人信息]}
{[诉讼请求]}
{[事实和理由]}
{[证据目录]}
{[结尾]}
```

### 详细模板

#### 当事人信息
```
原告：[姓名/名称]
住所地：[地址]
法定代表人：[姓名/职务]
联系方式：[电话/邮箱]

被告：[姓名/名称]
住所地：[地址]
法定代表人：[姓名/职务]
联系方式：[电话/邮箱]

第三人：[如有]
```

#### 诉讼请求
```
原告请求法院：

1. [具体诉讼请求一]

2. [具体诉讼请求二]

3. [其他请求]
```

#### 事实和理由

##### 一、案件事实
```
[详细陈述案件事实，包括时间、地点、人物、事件经过等]

1. [事实段落一]

2. [事实段落二]
```

##### 二、争议焦点
```
本案争议焦点为：[争议焦点内容]

关于此争议，[当事人陈述]：
[具体内容]
```

##### 三、法律依据
```
根据《[法律名称]》第[条款]规定：
[法律条文]

本案中，[法律分析]，
因此，原告的诉讼请求有充分的法律依据。
```

#### 证据目录
```
原告提交以下证据：

证据一：[证据名称]
证据来源：[来源]
证明目的：[证明内容]

证据二：[证据名称]
[同样格式]
```

#### 结尾
```
此致
[法院名称]

具状人：[签字]
[日期]

附件：
1. 本起诉状副本[X]份
2. 证据材料[X]份
```

## 代理词模板

### 基本结构

```
{[案件基本信息]}
{[争议焦点]}
{[对争议焦点的分析与意见]}
{[事实认定]}
{[法律适用]}
{[结论与建议]}
{[附：证据目录]}
```

### 详细模板

#### 案件基本信息
```
案件性质：[民事/刑事/行政]案件
案件编号：[案号]
审理法院：[法院名称]
审判员：[姓名]
委托代理人：[姓名，律师事务所]
```

#### 争议焦点
```
根据法庭调查和当事人陈述，本案争议焦点为：

1. [争议焦点一]

2. [争议焦点二]

3. [其他争议焦点]
```

#### 对争议焦点的分析与意见

##### 一、争议焦点一的分析
```
代理人认为：[总体意见]

具体分析如下：
[详细分析理由]
```

##### 二、争议焦点二的分析
```
[同样格式]
```

#### 事实认定
```
根据法庭调查的事实和相关证据，代理人认为：

1. [事实认定一]

2. [事实认定二]
```

#### 法律适用
```
本案应适用的法律：

1. 《[法律名称]》第[条款]规定：
[法律条文]

本案中，[法律分析]，应当适用此规定。

2. [其他法律依据]
```

#### 结论与建议
```
综上所述，代理人认为：

1. [总体结论]

2. 请求法庭：
   （1）[具体请求一]
   （2）[具体请求二]

请法庭依法支持原告的合理诉求。
```

#### 附：证据目录
```
代理人提交证据如下：

证据一：[证据名称]
证明目的：[证明内容]

[同样格式]
```

## 使用说明

Writer Agent在起草法律文书时：

1. **识别文书类型** - 根据场景自动判断需要哪种文书
2. **加载对应模板** - 从本知识库中读取相应模板
3. **智能填充** - 用案件信息替换模板中的占位符 {[内容]}
4. **格式校验** - 确保输出符合法律文书格式要求

### 占位符说明

- `{[内容]}` - 需要填充的具体内容
- `[民事/刑事/行政]` - 选择适用的选项
- `[姓名]` - 填入具体姓名
- `[日期]` - 填入具体日期

### 自定义说明

模板可根据具体案件类型进行调整：
- 民事案件使用民事程序模板
- 刑事案件使用刑事程序模板
- 行政案件使用行政程序模板

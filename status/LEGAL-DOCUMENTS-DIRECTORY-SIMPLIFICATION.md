## ✅ 使用指南

### 当前推荐结构
```
output/[案件编号]/
└── 04_法律文书/
    ├── (如需起诉状) → 创建"起诉状/"子目录
    ├── (如需答辩状) → 创建"答辩状/"子目录
    ├── (如需代理词) → 创建"代理词/"子目录

```

### 操作原则
1. **按需创建**: 只有实际需要时才创建子目录
2. **灵活命名**: 子目录名称根据实际文书类型确定
3. **避免冗余**: 不创建空目录
4. **保持清晰**: 目录结构简洁明了
